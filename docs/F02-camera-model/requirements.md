# F02: カメラモデル — 要求仕様書

## 目的

カメラの内部パラメータ（焦点距離、センサーサイズ、解像度、FOV）と外部パラメータ（位置、姿勢）をモデル化し、以下の機能を提供する。

- カメラのローカル座標系（前方・右方・上方ベクトル）の計算
- ワールド座標からカメラ座標への変換
- ワールド座標から画像座標（ピクセル座標）への投影
- 後続機能（F04視錐台、F05オクルージョン、F08角度スコア、F09投影スコア、F12配置パターン）が必要とするインターフェースの提供

## 入力パラメータ

### 内部パラメータ（固定仕様、デフォルト値あり）

| パラメータ | デフォルト値 | 単位 | 出典 |
|-----------|-------------|------|------|
| 焦点距離 | 3.5 | mm | LM3NC1M仕様 |
| センサー幅 | 5.76 | mm | DFK33GR0234仕様 (1/2.6型) |
| センサー高さ | 3.6 | mm | DFK33GR0234仕様 (1/2.6型) |
| 解像度（水平） | 1920 | px | DFK33GR0234仕様 |
| 解像度（垂直） | 1200 | px | DFK33GR0234仕様 |
| ピクセルサイズ | 3.0 | µm | DFK33GR0234仕様 |

FOVはセンサーサイズと焦点距離から自動計算する。明示的に設定する必要はないが、計算プロパティとして取得可能にする。

- 水平FOV: `2 * arctan(5.76 / (2 * 3.5))` ≈ 79°
- 垂直FOV: `2 * arctan(3.6 / (2 * 3.5))` ≈ 54°

### 外部パラメータ（カメラごとに異なる）

| パラメータ | 説明 | 単位 |
|-----------|------|------|
| position | カメラ位置 (x, y, z) | m |
| look_at | 注視点 (x, y, z) | m |
| up_hint | 上方向のヒントベクトル（デフォルト: [0, 0, 1] = Z軸上方） | 単位ベクトル |

**姿勢表現の方針**:
- ユーザーインターフェースとしては **注視点ベース（position + look_at）** を採用する。カメラ配置の文脈で最も直感的であり、「どこに置いてどこを見るか」で姿勢が自然に決まる。
- 内部的には注視点から **回転行列（3x3）** を計算して保持する。回転行列は座標変換の計算に直接使え、後続機能（F04視錐台の平面法線計算、F05レイキャスト、F08角度計算）で効率的。
- 最適化（F14-F15）で使う場合は、position (3DOF) + look_at (3DOF) = 6DOFの連続パラメータとして扱える。

## 出力（提供する機能）

1. **`forward` プロパティ**: カメラの前方方向ベクトル（単位ベクトル）。カメラから look_at 方向。
2. **`right` プロパティ**: カメラの右方向ベクトル（単位ベクトル）。
3. **`up` プロパティ**: カメラの上方向ベクトル（単位ベクトル）。
4. **`rotation_matrix` プロパティ**: ワールド座標系からカメラ座標系への回転行列 (3x3)。
5. **`world_to_camera(points) -> ndarray`**: ワールド座標をカメラ座標に変換。shape (N,3) → (N,3)。
6. **`project_to_image(points) -> ndarray`**: ワールド座標を画像座標（ピクセル）に投影。shape (N,3) → (N,2)。カメラ後方の点には NaN を返す。
7. **`hfov`, `vfov` プロパティ**: 水平/垂直画角（ラジアン）。
8. **`create_camera()` ファクトリ関数**: デフォルト内部パラメータでカメラを生成。

## 後続機能が必要とするインターフェース

| 後続機能 | F02に必要なもの |
|---------|---------------|
| F04 視錐台 | position, forward, right, up, hfov, vfov（視錐台の平面を構成するため） |
| F05 オクルージョン | position（レイの始点） |
| F08 角度スコア | position（2台のカメラと対象点から成す角度を計算するため） |
| F09 投影スコア | project_to_image（被写体の画像上サイズを推定するため）、resolution_w/h |
| F12 配置パターン | create_camera（position + look_at でカメラを定義） |

## 制約・品質基準

- 座標系: CLAUDE.md準拠（X=幅、Y=奥行、Z=高さ）
- カメラ座標系: X=右、Y=上、Z=前方（投影計算時にY軸を反転して画像座標に合わせる）
- numpy配列 shape (N,3) によるバッチ処理対応
- 単一点 shape (3,) も受け付ける（内部で (1,3) に変換）
- position と look_at が同一座標の場合は ValueError
- forward ベクトルと up_hint が平行の場合は ValueError
- レンズ歪みは無視する（TV歪曲 0.4% と小さいため）
