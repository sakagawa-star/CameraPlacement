# 全体計画

## Phase概要

### Phase 2：コアモデル（F01〜F06）
- 部屋・ベッド・カメラの3Dモデル定義
- カメラ視錐台（FOVピラミッド）の計算
- 空間のグリッド化と視認性判定

### Phase 3：カバレッジ評価（F07〜F11）
- 各グリッド点について何台のカメラから視認可能かを計算
- 三角測量角度・2D投影サイズの評価
- カバレッジマップの3D可視化

### Phase 4：配置パターンの比較（F12〜F13）
- 代表的な配置パターン（コーナー型、ハイブリッド型など）を定義
- カバレッジと品質指標で比較

### Phase 5：最適化（F14〜F15）
- 目的関数の定義
- 最適化アルゴリズムの適用（PSO、差分進化等）
