# F01: 空間モデル（部屋・ベッド） — 要求仕様書

## 目的

病室の3D空間（部屋・ベッド・カメラ設置可能領域）をAABB（軸平行境界ボックス）でモデル化し、点の包含判定機能を提供する。

※ レイ-AABB交差判定（オクルージョン）はF05で実装するため、F01のスコープ外。

## 入力パラメータ

| パラメータ | デフォルト値 | 単位 |
|-----------|-------------|------|
| 部屋の幅 (X) | 2.8 | m |
| 部屋の奥行 (Y) | 3.5 | m |
| 部屋の高さ (Z) | 2.5 | m |
| ベッド X範囲 | 0.9 〜 1.9 | m |
| ベッド Y範囲 | 1.5 〜 3.5 | m |
| ベッド Z範囲 | 0.0 〜 0.2 | m |
| カメラ設置 X範囲 | 0.2 〜 2.6 | m |
| カメラ設置 Y範囲 | 0.2 〜 3.3 | m |
| カメラ設置 Z範囲 | 0.2 〜 2.3 | m |

全パラメータはデフォルト値を持つが、変更可能とする。

## 出力（提供する機能）

1. **`is_inside_room(points) -> ndarray[bool]`**: 点が部屋内にあるか
2. **`is_on_bed(points) -> ndarray[bool]`**: 点がベッド領域内にあるか
3. **`is_valid_camera_position(points) -> ndarray[bool]`**: 点がカメラ設置可能領域内か

## 制約・品質基準

- 座標系: 原点=部屋の床面手前左コーナー、X=幅、Y=奥行、Z=高さ（CLAUDE.md準拠）
- 境界上の点は「内側」として扱う（<=, >= で判定）
- numpy配列 shape (N,3) によるバッチ処理対応（後続のカバレッジ計算で必要）
- 単一点 shape (3,) も受け付ける（内部で (1,3) に変換）
